Grid Containers

First, we can create two containers to wrap our page content. One that will expand 100% width and another that will limit the width to a set amount of pixels and center the content using auto margins

.container {
width: 960px;>
Margin: 0 auto;
}
.container-fluid {
width: 100%;
}

Grid Rows

Then, we can create row classes to act as vertical separation between rows of columns. These elements will reside inside of our containers and act as parent elements to columns that will reside inside of them.

row {
clear: both; /* Insures rows always stack vertically. */
}
/* apply clearfix to all rows to prevent them from collapsing their height */
.row:after {
content: ".";
display: block;
clear: both;
visibility: hidden; height: 0; line-height: 0;
}

Grid Columns

Next, we set all column classes to float in order to disply side by side nd give them a left margin (except on the first column in a row) if space is desired between your columns.

[class^="col-"] { float: left; margin-left: 2%; } [class^="col-"]:first-child { margin-left: 0; }

Grid Strategy

Lastly, we set widths for all the different size column possibilities. Let's assume a 12 column grid. Then simple math allows us to calculate the width each column should be. 100% divided by the number of columns minus any margin between them equals our column width.

Now we can make any layout we require by applying the class names:

<div class="container">
<div class="row">
<div class="col-4"> A </div>
<div class="col-4"> B </div>
<div class="col-4"> C </div>
</div>
</div>

This products a three column grid where each column takes up 1/3rd the width of the 12 column grid.

Width Formula

Column width= (row width - total margin / number of columns.
Assuming we are using IE box model.

.col-6 width = ( 100% - 2% ) / 2
.col-6 width = 98% / 2
.col-6 width = 49%
So:
.col-6 {width=49%;}

If using W3C:
column width= (row width -margin -padding -border ) / number of columns.
