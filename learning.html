Authority and Inheritance

If you use conflicting styles on the same element, descendent over rules ID, over rules Class, over rules * (universal) selector. The more specific a selector is, the more authrity is has!

When conflicts with equal authority arise, CSS will apply the last style that is closer to the bottom of the stylesheet ("last man" rule).

Certain CSS properties will be inherited from their parent elements.
